# CMake For VisualStudio Code

This extension provides support for [Cmake](http://www.cmake.org/) in [Visual Studio Code](https://code.visualstudio.com/) 0.10.0

![Screenshot](images/code.gif)

## Features

- Colorization
- Completion Lists 
- Snippets
- Quick Help
- Access To Online Help
